# sentry-demos/react

## Goal/Summary:
Show how Sentry works
- Import/Integrate
- Configuration
- Releases/SourceMaps

## Setup
1. Configure Raven with your `PUBLIC_DSN_KEY` in index.html
2. Setup and Deploy
```
$ npm install
$ npm run deploy
```
3. Go to http://localhost:3000 in your browser and begin throwing errors/events to Sentry!
